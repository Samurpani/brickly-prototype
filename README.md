# Brickly Prototype

This folder contains a lightweight starter project for your Brickly prototype.

It also now includes imported Bricly planning materials so product context lives in the repo alongside the prototype.

## Run locally

From this folder, start a simple local server:

```bash
npm start
```

Then open http://localhost:3000 in your browser.

## What’s included

- A clean landing page for your MVP concept
- Basic styling and a polished visual starting point
- A small JavaScript interaction for your first prototype flow
- Imported planning docs in `docs/source/`
- A concise working brief in `docs/ai-context.md`
- Imported HTML prototypes in `prototypes/`
- Supporting reference assets in `reference/`
